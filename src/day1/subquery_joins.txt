Question 3 ⭐⭐⭐

Show department names and the number of employees in each department, but only for departments whose average salary is greater than the company average salary.

Departments

| Column         | Description     |
| -------------- | --------------- |
| DepartmentID   | Primary Key     |
| DepartmentName | Department Name |

select d.departmentName , e.count(*) as emp_count,e.avg(salary) as avg_sal
from employees as e
inner join departments as d
on d.depratmentID=e.departmentID
groupby d.departmentName 
having avg_sal >( select avg(sal) from employess)

Question

Find employees whose salary is greater than the average salary of the department managed by David.

SELECT EmployeeName
FROM Employees
WHERE Salary >
(
    SELECT AVG(Salary)
    FROM Employees
    WHERE DepartmentID =
    (
        SELECT DepartmentID
        FROM Departments
        WHERE ManagerID =
        (
            SELECT EmployeeID
            FROM Employees
            WHERE EmployeeName = 'David'
        )
    )
);
